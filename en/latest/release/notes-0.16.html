Diofant 0.16

Not Released Yet

New features

Major changes

Compatibility breaks

  • Reorder tuples, returned by gcdex() and half_gcdex(), see #1454.

  • Removed Expr.lseries() (n=None option for series()), see #1418.

  • Conversion to exact domains now happens with full precision, see #1467.

Minor changes

  • Add as_integer_ratio(), see #1468.

Developer changes

Issues closed

See the release milestone for complete list of issues and pull requests involved in this release.
